# 📊 Data Analytics Learning Journey

Welcome to my **Data Analytics Learning Journey** repository. This repository documents my progress as I build practical data analytics skills through hands-on practice, real-world projects, and continuous learning.

## 🎯 Goal

Develop job-ready data analytics skills by mastering industry-standard tools and building a strong portfolio of projects.

---

# 🛠️ Skill Stack

### 📈 1. Excel / Google Sheets

* Advanced formulas:

  * VLOOKUP / XLOOKUP
  * INDEX + MATCH
  * SUMIF / COUNTIF / AVERAGEIF
  * IF & Nested IF
* Pivot Tables & Pivot Charts
* Data cleaning techniques

  * Text to Columns
  * Remove Duplicates
  * Conditional Formatting
* Dashboard creation and reporting

---

### 🗄️ 2. SQL

* Database fundamentals
* SELECT, WHERE, ORDER BY
* GROUP BY & HAVING
* JOINs
* Aggregate functions
* Common Table Expressions (CTEs)
* Window functions
* Query optimization

---

### 🐍 3. Python

* Python fundamentals
* NumPy
* Pandas
* Data cleaning & transformation
* Exploratory Data Analysis (EDA)
* Basic statistics
* Data visualization with Matplotlib & Seaborn

---

### 📊 4. Data Visualization

Focus on mastering **one** visualization tool deeply:

* Tableau *(preferred)* **or**
* Power BI

Topics include:

* Interactive dashboards
* KPIs & metrics
* Storytelling with data
* Business reporting

---

### 📐 5. Statistics Fundamentals

Learn the core concepts required for data analysis:

* Mean, Median & Mode
* Variance & Standard Deviation
* Probability basics
* Distributions
* Correlation
* Hypothesis testing
* Confidence intervals

---

### 💼 6. Portfolio Development

Build a portfolio showcasing real-world analytics skills:

* 3–4 end-to-end data analytics projects
* Clean documentation
* Well-structured GitHub repository
* Professional Resume
* Optimized LinkedIn profile
